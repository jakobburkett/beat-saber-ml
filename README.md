# steps i used to run the DDC stuff (demo)
1. Clone repo https://github.com/BradeyLounsbury/ddc_research
2. ```cd ddc_research```
3. ```python3 -m venv .venv```
4. ```source .venv/bin/activate```
5. ```pip3 install -r ./requirements.txt```
6. ```cd infer```
7. ```./docker_serve.sh```
8. ```deactivate``` when done
